# Comments

Réplique adds a full comment management system to your admin panel. Visitors can leave comments on any content — blog posts, products, pages, or any other model your site exposes — and you control everything from here.

## What you can do

From the admin panel you can:

- **Browse all comments** across every content type in one place, grouped by record
- **Moderate** individual comments — approve, reject, or mark as spam
- **Bulk-moderate** a selection of comments in a single action
- **Reply** to any comment directly from the admin, attributed to your admin account
- **Pin** a comment so it always appears first in the thread
- **Block IP addresses** to prevent persistent spammers from submitting further comments
- **View the full comment thread** above any reply, so you have context when moderating

## Where to find it

Open the **Comments** section in the admin sidebar. If your panel has been configured with a custom navigation group, look for it there instead.
